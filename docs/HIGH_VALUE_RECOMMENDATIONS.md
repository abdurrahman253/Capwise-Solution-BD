# Capwise High-Value Website Roadmap

The codebase now has the foundation for a premium advisory platform. The following items create the largest client and business value, but require approved inputs or operational decisions.

## Credibility and conversion

- Publish 2–4 evidence-backed case studies with permission, measurable outcomes and a defined scope.
- Obtain written testimonials and logo permissions. Never use fabricated evidence.
- Use verified professional team photography and approved credentials.
- Add a consultation triage dashboard so every enquiry has an owner, status, next action and response target.
- Create industry-specific consultation landing pages for foreign investors, RMG/manufacturing and import-export businesses.

## Regulatory knowledge system

- Assign an internal reviewer for NBR, RJSC, BIDA, Bangladesh Bank and labour updates.
- Add a content workflow: draft, source verified, technical review, legal review where needed, approved, published, review due, withdrawn.
- Show reviewer, official source, affected audience and last-reviewed date on every time-sensitive item.
- Build a compliance calendar generator only after the underlying rules and maintenance owner are approved.

## Resource and lead system

- Finish the three approved starter PDFs with version date, sources and disclaimer.
- Connect guide interest and newsletter data to a CRM or MongoDB admin queue.
- Add consent records, unsubscribe handling and download/release analytics.
- Use resources in topic clusters that lead naturally to the relevant service and consultation page.

## Client experience

- Add a real portal for request status, secure document requests and messages after roles and retention rules are approved.
- Add appointment scheduling only when Capwise confirms adviser calendars and response rules.
- Create a post-consultation onboarding sequence: scope summary, document checklist, engagement status and named contact.
- Add a client-facing service timeline with dependencies rather than guaranteed completion dates.

## SEO, local trust and measurement

- Complete Google Business Profile, confirmed map pin, official Facebook and LinkedIn pages.
- Connect Search Console, Bing Webmaster Tools and approved privacy-conscious analytics.
- Build service/topic clusters with reviewed original content and internal links.
- Track consultation conversion by page, service, guide and campaign without collecting unnecessary sensitive data.
- Schedule quarterly content, broken-link, accessibility, performance and regulatory-review audits.

## Operations, security and handover

- Use verified domain email, SPF, DKIM and DMARC.
- Enable shared rate limiting and database persistence before high-traffic campaigns.
- Define retention, deletion, breach-response and access-review procedures.
- Add automated preview deployment checks, lint, build and route audits in GitHub Actions.
- Prepare a content-owner guide, environment-variable guide, incident contacts and launch rollback plan.
